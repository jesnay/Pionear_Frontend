# Stadtspaziergang Heidelberg

## Installation

1. clone Repository

```
git clone https://github.com/stjesnay/P3_FYJ.git
cd P3_FYJ/
```

2. install and start Backend

```
cd backend/
npm install
npm start
```

⚠️ DONT EXIT BACKEND

3. install and start Frontend (in new Terminal)

```
cd frontend/
npm install
npm start
```
